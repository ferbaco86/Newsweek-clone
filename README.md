# Newsweek-Clone
A responsive clone of the main webpage of Newsweek (https://www.newsweek.com/) created focusing on structuring and positioning elements using Bootstrap 4 as the main framework.

## Built With

- HTML
- CSS
- Bootstrap 4
- Responsive Design
- Fontawesome
